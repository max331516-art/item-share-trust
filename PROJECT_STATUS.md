# RentIt — Статус проекта

> P2P-сервис аренды вещей между соседями.  
> Платформа выступает как посредник, платёжный агент и арбитр.

## 🔴 Не реализовано (план)

### Этап 1 — База (критично)
- [ ] **База данных**: таблицы profiles, items, rentals, reviews, disputes, rental_photos
- [ ] **Аутентификация**: регистрация/вход по Email + пароль, подтверждение email
- [ ] **Профиль пользователя**: имя, аватар, телефон, рейтинг
- [ ] **CRUD объявлений**: создание, редактирование, удаление, загрузка фото
- [ ] **Каталог из БД**: реальные данные вместо мок-данных

### Этап 2 — Бронирование
- [ ] **Календарь доступности** на странице вещи
- [ ] **Процесс бронирования**: выбор дат → подтверждение владельцем → аренда
- [ ] **Имитация оплаты**: UI оплаты с "блокировкой" депозита (без реального списания)
- [ ] **Статусы аренды**: pending → confirmed → active → completed / disputed
- [ ] **Фотофиксация**: загрузка фото "до" и "после" передачи вещи

### Этап 3 — Доверие и коммуникация
- [ ] **Система отзывов**: двусторонние отзывы после аренды (арендатор ↔ владелец)
- [ ] **Споры/диспуты**: механизм открытия спора с доказательствами (фото)
- [ ] **Чат** между арендатором и владельцем (реалтайм)
- [ ] **Уведомления**: о новых бронированиях, сообщениях, статусах

### Этап 4 — Личный кабинет
- [ ] **Мои объявления**: список, статистика просмотров
- [ ] **Мои аренды**: текущие, прошлые, как арендатор и как владелец
- [ ] **Настройки профиля**: редактирование данных, аватар
- [ ] **История транзакций**: платежи, депозиты, комиссии

### Этап 5 — Продвинутые фичи
- [ ] **Геолокация**: показ вещей рядом с пользователем на карте
- [ ] **Модерация**: проверка объявлений перед публикацией
- [ ] **Верификация**: подтверждение телефона (SMS)
- [ ] **SEO**: мета-теги, JSON-LD для товаров, OG-изображения
- [ ] **Аналитика**: статистика для владельцев (просмотры, конверсия)

## 🗂 Структура проекта

```
src/
├── assets/              # Изображения (hero)
├── components/
│   ├── ui/              # shadcn/ui компоненты
│   ├── Navbar.tsx        # Навигация
│   ├── Footer.tsx        # Футер
│   ├── ItemCard.tsx      # Карточка вещи
│   └── NavLink.tsx       # Ссылка навигации
├── hooks/               # Кастомные хуки
├── integrations/
│   └── supabase/        # Клиент и типы БД (авто-генерация)
├── lib/
│   ├── mock-data.ts     # Мок-данные (будут заменены на БД)
│   └── utils.ts         # Утилиты
├── pages/
│   ├── Index.tsx         # Главная
│   ├── Catalog.tsx       # Каталог
│   ├── ItemDetail.tsx    # Детальная страница вещи
│   ├── CreateListing.tsx # Создание объявления
│   ├── HowItWorks.tsx    # Как это работает
│   └── NotFound.tsx      # 404
├── App.tsx              # Маршрутизация
├── App.css              # Стили
├── index.css            # Дизайн-токены, темы
└── main.tsx             # Точка входа
```
